基于Hexo搭建本地博客并部署到云服务器教程

Created2023-11-06|全栈
|Word count:643|Reading time:2min|Post View:

1.安装依赖工具包
yum install -y curl-devel expat-devel gettext-devel openssl-devel zlib-devel
2.安装编译工具
yum install -y gcc perl-ExtUtils-MakeMaker package
3.删除旧版本的git
yum remove git -y
4.进入home目录
cd
5.下载git到home目录
wget https://www.kernel.org/pub/software/scm/git/git-2.34.0.tar.gz –no-check-certificate
yum install wget -y
6.解压文件
tar -zxf git-2.34.0.tar.gz
7.安装git
进入git目录
cd git-2.34.0
编译
make all prefix=/usr/local/git
安装git到/usr/local/git下
make install prefix=/usr/local/git
8.配置git环境变量
将git加入到path目录
echo ‘export PATH=$PATH:/usr/local/git/bin’ >> /etc/bashrc
使配置生效
source /etc/bashrc
查看git版本
git –version

3.创建用户并设置免密登录
1.创建用户并设置密码
adduser yu
passwd yu
2.为用户分配权限
把用户加入到wheel组
usermod yu -G wheel
3.本地创建密钥
在本机电脑终端输入
ssh-keygen -t rsa
key的保存路径

4.配置公钥
回到服务器终端，切换用户
su - yu
创建.ssh文件夹
mkdir .ssh
新建authorized_keys
vim .ssh/authorized_keys
如果提示没有vim则先安装
sudo yum install -y vim
将刚刚id_rsa.pub公钥中的内容，复制粘贴到文件里
查看.ssh/authorized_keys
cat .ssh/authorized_keys
退出当前用户

5.git仓库配置
退出当前用户
exit
创建git目录并修改目录的所有权和用户权限
mkdir /home/git
chown -R yu:yu /home/git/
chmod -R 755 /home/git/
进入目录，建立git仓库，修改权限
cd /home/git
git init –bare LbjYuBlog.git
chown yu:yu -R LbjYuBlog.git
新建构子文件 post-receive
vim /home/git/LbjYuBlog.git/hooks/post-receive
进入文本编辑器，粘贴下面两行
#!/bin/nash
git –work-tree=/home/LbjYuBlog –git-dir=/home/git/LbjYuBlog.git checkout -f
查看文件
cat –work-tree=/home/LbjYuBlog –git-dir=/home/git/LbjYuBlog.git checkout -f
修改文件权限
chmod +x /home/git/LbjYuBlog.git/hooks/post-receive

6.nginx配置
mkdir /home/LbjYuBlog/
chown -R yu:yu /home/LbjYuBlog/
chmod -R 755 /home/LbjYuBlog/
安装nginx
yum install -y nginx
启动nginx
systemctl start nginx.service
查看nginx服务状态，绿色的active说明启动成功
systemctl status nginx.service

查看nginx的默认配置文件的安装位置
nginx -t


配置
保存后重启nginx服务
systemctl restart nginx.service

7.在hexo站点配置文件中配置
在_config.yml中修改
deploy:
type: git
repository: yu@106.52.91.121:/home/git/LbjYuBlog.git
branch: master
在本地hexo博客根目录下，打开终端，部署
hexo clean
hexo g
hexo d
